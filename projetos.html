Apresentação didatica do MySQL

projeto que desenvolvi sobre MySQL, uma das ferramentas mais poderosas e amplamente utilizadas no gerenciamento de bancos de dados. Utilizei o Canva para criar esta apresentação, trazendo uma abordagem visual que facilita o entendimento dos principais conceitos e funcionalidades do MySQL.

Criação de um site ultilizando HTML/CSS

projeto em que desenvolvi um site do zero, utilizando as linguagens HTML e CSS. O objetivo deste trabalho é mostrar como essas tecnologias, que são a base da web, podem ser utilizadas para criar um site completo, com estrutura, estilo e responsividade.

Criação de um Site Utilizando HTML Puro

projeto desenvolvido utilizando HTML puro, sem o uso de outras linguagens de programação ou frameworks. Este projeto tem como objetivo mostrar como o HTML, que é a base da construção de qualquer página da web, pode ser utilizado para criar a estrutura de um site funcional e organizado
